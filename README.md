# E-Commerce Product 

This project is a part of Front End Mentor Challenges, I used **Vue 3**, **Vite** and **Tailwind CSS** to build this project.

## Setup

```sh
  
  pnpm install
  
```

## Start


```sh

  pnpm dev
  
```
